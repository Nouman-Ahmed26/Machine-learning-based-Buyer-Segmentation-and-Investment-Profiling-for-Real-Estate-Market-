# 🏠 Parcl - ML Based Buyer Segmentation & Investment Profiling

## 📌 Project Overview
Machine Learning based Buyer Segmentation and Investment 
Profiling for Real Estate Market Intelligence built for 
Parcl Co. Limited.

## 🎯 Problem Statement
Parcl lacked a data-driven understanding of different buyer 
types, investment motivations, and geographic differences 
in buyer behavior.

## 💡 Solution
An AI-driven segmentation system using K-Means clustering 
that groups buyers into 4 distinct segments:
- 🌍 Global Investors
- 🏠 First-Time Buyers
- 🏢 Corporate Buyers
- 💎 Luxury Investors

## 📊 Datasets
- **Clients** — 2,000 buyer records
- **Properties** — 10,000 property transactions

## 🛠️ Tech Stack
- Python, Pandas, NumPy
- Scikit-learn (KMeans Clustering)
- Plotly (Interactive Charts)
- Streamlit (Dashboard)

## 🖥️ Dashboard Pages
1. 🏠 Home Dashboard — KPIs & overview
2. 👥 Buyer Segmentation — Clusters & elbow method
3. 📈 Investor Behavior — Investment patterns
4. 🗺️ Geographic Analysis — Country & region maps
5. 🔍 Segment Insights — Per segment deep dive

## 🚀 How to Run
pip install -r requirements.txt
streamlit run app.py
